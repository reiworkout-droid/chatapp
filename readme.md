## 制作アプリのタイトル   マッチョングアプリ. 

## 制作アプリの説明（40文字程度）. 
form側で登録したトレーナーをプラットフォームで一覧表示し、ユーザーが自分に合ったトレーナーを選択することができるアプリ. 

## 工夫した点・こだわった点. 
- フォームで入力した内容をfirebaseに保存、表示. 
- 画像はCloudinaryでURLに変換し、URLをfirebaseに保存、表示. 
- フォームとプラットフォームを別ページに分ける. 

## 次回トライしたこと（または機能）. 
- 別ページで口コミをユーザーが登録でき、それをfirebaseに保存.

## フリー項目（感想、シェアしたいこと等なんでも）. 
画像をURLに変換する方法を調べるのにも実装するのにもかなり時間を要したが、実現でき、内容についても理解ができたので大満足です。
