制作アプリのタイトル   Fit Hub

制作アプリの説明（40文字程度） 
・筋トレメモ記録と体重＋BMIを管理するプラットフォーム

工夫した点・こだわった点 
[Fit Hub]
・日付のセレクトボックスを繰り返しで実装
・種目、重量、回数、RMを全てローカルストレージに保存
・プッシュした数値がローカルストレージの最大値より大きい時にテキスト挿入
・＋ボタン押印でセット数を追加、種目数を追加
・入力した値を配列及びオブジェクトを用いて記録

[Weight Manager]
・配列のデータをローカルストレージに保存
・グラフの作成
・グラフの値をローカルストレージに保存
・更新後もグラフが残るように設定

次回トライしたこと（または機能） 
・カレンダーの日付を選択すると該当日に筋トレメモを追加できる。
・APIを用いた栄養管理ツール

フリー項目（感想、シェアしたいこと等なんでも） 　
今回は実際に卒業制作で実装したい機能を作成してみた。
まだまだ不完全なのは否めないが、現段階でこの２ツールはある程度形になることがわかった。
